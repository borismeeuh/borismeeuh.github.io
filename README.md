# Getting Started with Create React App

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm build`

Runs the build process, including the custom webpack.config.js.

## Git structure

The repo consists of three branches: develop, master and gh-pages. Once changes are merged from develop to master, 
the changes will also automatically be merged to gh-pages.
